# 🚀 Sowaka 页面快速启动指南

## 📋 一分钟快速启动

### 方法一：命令行启动

1. **进入项目目录**
```bash
cd web-project
```

2. **安装依赖（首次运行）**
```bash
npm install
```

3. **启动开发服务器**
```bash
npm run dev
```

4. **访问页面**
- 打开浏览器访问：`http://localhost:5173/sowaka`

### 方法二：从其他页面导航

在任何 Vue 组件中添加链接：

```vue
<router-link to="/sowaka">前往 Sowaka</router-link>
```

## 🐛 常见问题快速解决

### 问题：页面显示空白
**解决方案：**
```bash
# 确保依赖已安装
npm install

# 重新启动开发服务器
npm run dev
```

### 问题：样式显示不正常
**解决方案：**
- 清除浏览器缓存（Ctrl + Shift + R）
- 检查浏览器控制台是否有错误信息

### 问题：端口被占用
**解决方案：**
```bash
# 方法1: 使用不同端口
npm run dev -- --port 3000

# 方法2: 杀掉占用端口的进程
# Windows: netstat -ano | findstr :5173
# Linux/Mac: lsof -ti:5173 | xargs kill
```

### 问题：移动端菜单不工作
**解决方案：**
- 确保浏览器窗口宽度 ≤ 768px
- 检查浏览器开发者工具是否启用了移动设备模拟

## 📱 最佳查看方式

### 桌面端
- **推荐分辨率**: 1920x1080 或更高
- **推荐浏览器**: Chrome、Firefox、Edge（最新版本）
- **最佳缩放**: 100%

### 移动端
- **推荐宽度**: 375px - 768px
- **测试方式**: 浏览器开发者工具 → 移动设备模拟
- **推荐设备模拟**: iPhone 12 Pro, iPad
