# 极简日式风格页面说明

## 页面概述

这是一个参考日式极简美学设计的页面，具有以下特点：

### 设计风格
- **左侧区域**：大幅天空渐变背景，营造宁静氛围
  - 垂直排列的日文标题"白夜"
  - 柔和的云朵动画效果
  - 装饰性日文文字

- **右侧区域**：极简主义卡片展示
  - 三张优雅的卡片，带有斜线纹理
  - 米白色调，柔和阴影
  - 精致的排版和留白

### 技术特点
1. **渐变背景**：使用CSS渐变模拟天空色彩
2. **动画效果**：
   - 云朵浮动动画
   - 卡片淡入动画
   - 悬停提升效果
3. **响应式设计**：适配不同屏幕尺寸
4. **优雅排版**：使用日文字体和西文衬线字体

## 访问方式

启动开发服务器后，访问：
```
http://localhost:5173/minimalist
```

## 自定义建议

### 修改标题
在 `MinimalistPage.vue` 中修改：
```vue
<h1 class="main-title">白夜</h1>
<p class="subtitle">—Midnight Sun—</p>
```

### 修改卡片内容
修改卡片标题和文字：
```vue
<h2 class="card-title">H O 1</h2>
<p class="card-text">人　間　不　信</p>
```

### 调整颜色
修改渐变背景色：
```css
background: linear-gradient(
  to bottom,
  #a8c5dd 0%,    /* 顶部蓝色 */
  #d4b5a8 30%,   /* 粉色过渡 */
  #e8d5c4 60%,   /* 米色 */
  #c5d3e0 100%   /* 底部蓝灰色 */
);
```

### 添加更多卡片
复制卡片HTML结构并修改内容：
```vue
<div class="minimal-card">
  <div class="card-pattern"></div>
  <div class="card-content">
    <h2 class="card-title">H O 4</h2>
    <p class="card-text">新　内　容</p>
  </div>
</div>
```

## 字体说明

页面使用以下字体（通过Google Fonts加载）：
- **Noto Serif JP**：日文衬线字体
- **Cinzel**：优雅的西文衬线字体

如需使用这些字体，请在 `index.html` 中添加：
```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Noto+Serif+JP:wght@300;400&family=Cinzel:wght@300;400&display=swap" rel="stylesheet">
```

## 设计理念

这个页面体现了日式美学的核心原则：
- **侘寂（Wabi-Sabi）**：简约而不失优雅
- **余白（Yohaku）**：大量留白营造呼吸感
- **静寂（Seijaku）**：宁静的色彩和动画
- **精致（Seimitsu）**：细节处理考究

## 应用场景

- 文学作品展示
- 艺术品集合
- 个人作品集
- 品牌形象页面
- 章节导航页面
