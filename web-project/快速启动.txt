❓ 遇到问题？

1. 页面白屏
   → 检查是否运行了 npm install
   → 查看浏览器控制台错误信息

2. 图片/视频不显示
   → 确认已运行 COPY_ASSETS.bat
   → 检查 public/ 目录是否有媒体文件

3. npm 命令不可用
   → 安装 Node.js (v20.19.0+ 或 v22.12.0+)
   → 重启命令行窗口

4. 端口被占用
   → 修改端口: npm run dev -- --port 3000
