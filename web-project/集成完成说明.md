## 📝 注意事项

### ⚠️ 重要提醒

1. **媒体文件必须复制**
   - 运行 `COPY_ASSETS.bat` 或手动复制
   - 否则图片和视频无法显示

2. **Node.js版本要求**
   - Node.js >= 20.19.0 或 >= 22.12.0
   - 使用 `node -v` 检查版本

3. **浏览器兼容性**
   - 推荐使用 Chrome、Edge、Firefox
   - 需要支持 ES6+ 和 Canvas API

4. **开发模式 vs 生产模式**
   - 开发: `npm run dev` (热更新)
   - 生产: `npm run build` (优化构建)
