# 🚀 快速启动指南

## 前提条件

确保已安装：
- ✅ Node.js (v20.x 或更高)
- ✅ MySQL (v8.x 或更高)
- ✅ npm 或 yarn

## 步骤 1: 安装依赖

```bash
cd backend
npm install
```

## 步骤 2: 启动 MySQL

确保 MySQL 服务正在运行：

**Windows:**
```bash
# 检查 MySQL 服务状态
net start MySQL80

# 如果未启动，启动服务
net start MySQL80
```

**macOS/Linux:**
```bash
# 检查状态
sudo systemctl status mysql

# 启动服务
sudo systemctl start mysql
```

## 步骤 3: 配置数据库

编辑 `config.js` 文件，修改数据库密码：

```javascript
module.exports = {
  database: {
    host: 'localhost',
    port: 3306,
    user: 'root',
    password: '',  // ← 修改为你的MySQL密码
    database: 'novel_reading_db'
  }
};
```

## 步骤 4: 初始化数据库

```bash
npm run init-db
```

看到以下输出表示成功：
```
✅ 已连接到MySQL服务器
✅ 数据库 "novel_reading_db" 已创建或已存在
✅ 表 "contents" 已创建
✅ 已插入 7 条示例数据
✨ 数据库初始化完成！
```

## 步骤 5: 启动后端服务器

```bash
# 开发模式（推荐）
npm run dev

# 或生产模式
npm start
```

看到以下输出表示成功：
```
==================================================
🚀 服务器运行在: http://localhost:3000
📝 环境: development
🗄️  数据库: localhost:3306
==================================================
```

## 步骤 6: 测试 API

打开浏览器访问：
- http://localhost:3000 - 查看 API 信息
- http://localhost:3000/health - 健康检查
- http://localhost:3000/api/contents - 获取所有内容

或使用命令行测试：
```bash
curl http://localhost:3000/api/contents
```

## 步骤 7: 启动前端项目

在新的终端窗口中：

```bash
cd ../web-project
npm install
npm run dev
```

前端将在 http://localhost:5173 启动。

访问 http://localhost:5173/sowaka 查看页面效果。

## 🎉 完成！

现在你应该看到：
- ✅ 后端 API 运行在 3000 端口
- ✅ 前端应用运行在 5173 端口
- ✅ SowakaPage 页面从数据库加载并显示内容

## ⚠️ 常见问题

### 问题 1: MySQL 连接失败

**错误信息：** `ER_ACCESS_DENIED_ERROR: Access denied for user 'root'@'localhost'`

**解决方法：**
1. 检查 `config.js` 中的密码是否正确
2. 尝试使用 MySQL 命令行登录验证：
   ```bash
   mysql -u root -p
   ```

### 问题 2: 端口被占用

**错误信息：** `Error: listen EADDRINUSE: address already in use :::3000`

**解决方法：**
1. 修改 `config.js` 中的端口号
2. 或停止占用该端口的程序

### 问题 3: 前端无法连接后端

**错误信息：** `Network Error` 或 CORS 错误

**解决方法：**
1. 确保后端服务器正在运行
2. 检查 `config.js` 中的 CORS 配置
3. 确认前端 API_BASE_URL 正确
