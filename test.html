使用 CSS 与 JS 实现优雅的打字机效果

DATE: 2026-03-02 | TAG: FRONTEND, ANIMATION | AUTHOR: V. EVERGARDEN

一、 核心原理与构思

要实现打字机效果，其核心在于控制文本的逐步显现。我们可以通过 CSS 的 @keyframes 动画结合 steps() 函数，或是借助 JavaScript 进行精准的 DOM 操作来实现。前者轻量且优雅，后者则能适应更复杂的业务逻辑。

二、 CSS 纯享版实现

若您的文本内容是固定的，我推荐您使用 CSS 的方式。它就像是已经打好孔的机械乐谱，执行起来分毫不差。以下是基础的实现代码片段：

/* 设定文本容器的溢出隐藏与光标闪烁 */
.typewriter {
  overflow: hidden;
  border-right: .15em solid var(--gold);
  white-space: nowrap;
  margin: 0 auto;
  letter-spacing: .15em;
  animation:
    typing 3.5s steps(40, end),
    blink-caret .75s step-end infinite;
}

/* 文本展开动画 */
@keyframes typing {
  from { width: 0 }
  to { width: 100% }
}

/* 光标闪烁动画 */
@keyframes blink-caret {
  from, to { border-color: transparent }
  50% { border-color: var(--gold); }
}

在上述代码中，steps() 函数是灵魂所在。它打破了传统动画的平滑过渡，强制将其分为离散的步骤，从而完美模拟了机械打字机“咔哒、咔哒”逐字敲击的视觉特征。
