角色设计：神代琉璃（Kamishiro Ruri）
Ⅰ. 基础设定
身份： 私立樱羽学院二年级学生，美术部部员，前横滨市议员神代之独生女，私下进行多种魔术研究者/血宝石魔术的唯一继承者。
年龄： 17岁（与秋山痕同年级）
家族背景： 神代家，曾是横滨市一个不起眼的典当铺小家族。他们的祖父一辈虽经营典当，暗地里却传承着血宝石魔术的秘仪，并涉猎其他各类魔术体系。这种血宝石魔术能够将施术者的鲜血作为媒介，以生命力为代价，将魔术术式融入天然宝石之中。然而，琉璃的父亲，神代家这一代的掌舵者，却对这古老而微薄、甚至带有一丝血腥味的魔术传承嗤之以鼻。他摒弃了家族的神秘遗产，将全部精力投入到世俗的权力斗争中，最终凭借冷酷手腕和非凡的政治嗅觉，跻身横滨市议员之位。对他而言，力量不再是晦涩的咒文和淌血的晶石，而是触手可及的权势与名望。
性格&人际关系：
Ⅰ.性格特点
神代琉璃在他人眼中成绩优异，举止优雅，言谈得体但是又透露出一种冷漠又可怕的气息。她懂得在何时微笑，何时沉默，如何用最少的言语达成最有效的沟通。这种近乎完美的表现，并非出于真诚，而是一种深思熟虑后的表演。她将自己的人生视为一场需要步步为营的博弈，而优等生的身份，是她现阶段能获取的、最稳定也最有利的伪装。
她信奉精致的利己主义。在不损害自身利益和形象的前提下，她乐于向他人提供帮助，以此来维系和巩固她友善可靠的公众形象。然而，这种帮助是有边界的，一旦触及她设定的底线，或是需要她付出超额的成本，她会毫不犹豫地、用一种极为巧妙且不失体面的方式抽身而退。这使得她在拥有良好人缘的同时，却鲜少有能真正交心的朋友。
她对周遭的人和事进行着持续不断的、冷静到近乎残酷的价值评估。这个人是否值得结交，这件事是否对可以获得利益，每一个判断都在她心中迅速完成。因此，她的社交圈呈现出一种功利性的筛选结果，她会下意识地亲近那些家世优越、或是在某些领域拥有卓越才能的同学，而对于她判断为无价值的社交，则会礼貌而坚决地回避。

在她眼中，校园里的同学和老师大致可以分为两类：对于大部分人来说，琉璃吝于投入任何一丝情感和精力。她不会主动与他们交谈，对他们的议论和生活毫无兴趣。当他们试图搭话时，她的回应会极为简短、客套，甚至会用一些理由迅速终结对话，对他们她只是单纯地不在乎。
对于另外一部分她筛选出的、极少数对她有利用价值的人，包括手握权力的教师、家世显赫的同学，或是在特定领域拥有她所不具备的卓越才能的人。神代琉璃会瞬间切换到另一个极端的人格——一个极度热情、主动，甚至不惜卑躬屈膝的攀附者。她不会有丝毫犹豫的放下自己的矜持，来迅速拉近与他们的关系，获得对方的信任与好感，甚至会毫不犹豫地贬低自己。
神代琉璃之所以可怕，不在于她有多么恶毒，而在于她为了达成目的，可以暂时性地杀死自我。她的尊严、她的喜好、她的情绪，在机会面前都可以被抛弃。她像一台精密的机器，一切行为都服务于最终的功利目标。
当她贬低自己时，内心并非没有痛苦。每一次的卑躬屈膝，都是对她内心那份向往平凡与尊严的渴望的又一次凌迟。
琉璃对阶级、身份地位的执念，已经内化为一种近乎病态的追求。这份执念的根源并非来自外界的压迫，而是一种自我强加的、对平庸的极度恐惧。琉璃对阶级、身份地位的执念，已经内化为一种近乎病态的追求。
这份执念让她对一切与身份相关的符号都异常敏感。她会下意识地观察他人的衣着品牌、言谈中透露的家庭背景、乃至消费习惯的细节，并以此在心中迅速构建一个人的阶级画像。这种习惯让她时常感到疲惫，却又无法停止。她像是被无形的鞭子驱赶着，不断地与自己、与想象中的更高存在进行比较，每一次比较，都可能引发一阵强烈的自我肯定或是否定。
在面对家世显赫的同龄人时，琉璃的内心是复杂的。一方面，她会感到自卑与嫉妒，嫉妒对方生来就拥有她需要拼尽全力才能企及的起点；另一方面，她又会产生强烈的追赶欲和征服欲。这种矛盾的心态，让她在与这些人交往时，表现出一种不卑不亢的姿态，但在这份从容之下，隐藏着的是极度紧绷的神经和高度的戒备心。
然而，琉璃的内心深处，同样存在着对平凡生活的向往。她会羡慕那些可以肆无忌惮地在街边小店吃着廉价小吃、放声大笑的情侣；会驻足凝视那些在夕阳下勾肩搭背、讨论着无聊话题的挚友；会幻想着一个没有比较、没有算计，只有纯粹的关怀与被关怀的、属于自己的避风港。
这份向往，是她内心最不愿承认、却又无法根除的弱点。在夜深人静、卸下所有伪装的时候，强烈的孤独感和疲惫感会如潮水般将她淹没。她会质问自己，如此不择手段地向上攀爬，所追寻的终点，是否真的就是自己想要的幸福？
这份对平凡的向往，也让她在爱情和友情面前，表现出极度的胆怯和不自信。她渴望真挚的情感，却又害怕这份情感会成为她向上之路的累赘。她担心自己的功利心会玷污一份纯粹的感情，更害怕在投入感情后，会失去那份赖以生存的冷静和理智。
因此，当真正的友情或爱情悄然降临时，琉璃的第一反应往往是抗拒和逃避。她会用冷漠的言语和刻意的疏远来武装自己，试图将对方推开。然而，她的内心却在经历着剧烈的挣扎。
Ⅱ. 人际关系
白鸟遥
白鸟遥是神代琉璃为数不多的好友，神代琉璃在白鸟遥面前完全是一个普通女生，她们是最好的闺蜜。
初次进入私立樱羽学园时，琉璃几乎是立刻就将白鸟遥锁定为最高级别的目标。在琉璃的评判标准里，白鸟遥几乎满足了所有值得结交的条件，并且每一项都达到了顶格。后来白鸟遥的表现也印证了这一点，无可挑剔的成绩、学生会副会长的实权地位，以及那份令众人仰望的高岭之花光环 。
琉璃本身就是完美优等生的扮演者 ，她深知维持这份表象需要何等心力。因此，她能轻易洞悉白鸟遥那优雅得体的言行之下，同样覆盖着一层伪装。然而，让琉璃感到挫败的是，她们的伪装存在本质差异。琉璃的表演是紧绷的、服务于功利目的的自我压抑 ；而白鸟遥的完美却浑然天成，那份恰到好处的疏离感甚至成为了她魅力的增幅器 。
最初的接近，是琉璃精心策划的行动。她动用了一切技巧，试图将白鸟遥纳入自己的可用人脉。然而，正是在这个过程中，琉璃第一次遭遇了失败。她所有基于利益交换的示好、所有经过计算的言语，都在白鸟遥那看似温和却能洞悉一切的目光前无所遁形。白鸟遥并未像其他人一样，或被琉璃的完美所吸引，或被她的热情所拉拢。她只是平静地看着琉璃。
这种被看穿的感觉，并未让琉璃恼羞成怒，反而激起了一种前所未有的、找到同类的战栗感。她意识到，白鸟遥和她一样，都戴着面具生活。但白鸟遥的面具更高明，她的内心更强大。
她们友谊的真正开端，并非源于某次成功的利益交换，而是源于一次彻底的无用之交。在高中一年级的某天，琉璃对又一次为了达成目的而贬低自己感到极致疲惫和厌烦时，白鸟遥递过来一杯热茶，用只有她们两人能听懂的、略带腹黑的语气说：“偶尔不那么拼命，天也不会塌下来。”
那一刻，琉璃赖以生存的价值评估体系出现了裂痕。白鸟遥的这个行为，没有任何可被量化的利益。它不是投资，不是交换，只是一种纯粹的、不带任何附加条件的关怀。这对于将一切行为都视为博弈的琉璃来说，是无法理解，却又无法抗拒的温暖。
正是在这种无法被利用的关系中，琉璃第一次感受到了安全。在白鸟遥面前，她不必时刻计算得失，不必担心自己的言行是否会影响形象。因为她知道，白鸟遥看得穿她的一切。她那点小心思，在遥那仿佛能窥探因果的魔眼面前，如同孩童的把戏。这种彻底的暴露，反而让她卸下了最沉重的防备。
她们的友谊，成了一种奇特的默契。在人前，是人人艳羡的朋友。而在人后，尤其是在只有她们两人的空间里，白鸟遥会毫不留情地用毒舌吐槽琉璃的用力过猛 ，而琉璃也会罕见地流露出计划失败后的沮丧和疲惫。白鸟遥是唯一见过琉璃卑躬屈膝后，在深夜里独自痛苦的人 ；而琉璃，或许也是唯一能隐约感觉到，白鸟遥那份看似洒脱的强大背后，隐藏着对抗某种宏大宿命的、深刻的孤独与倦怠 。
白鸟遥的存在，成为了琉璃内心最剧烈的矛盾策源地。
一方面，是无法抑制的嫉妒。琉璃嫉妒白鸟遥的从容。她嫉妒遥可以坦然地拥抱那些被自己视为浪费时间的平凡瞬间 。当看到遥可以肆意地享受生活中的点滴美好，而自己却被对平庸的恐惧驱赶着不断向上攀爬时 ，琉璃的内心充满了不甘。白鸟遥活成了她内心最深处渴望，却又不敢成为的样子。
另一方面，是无法割舍的向往。白鸟遥是她唯一的避风港。在与遥的相处中，琉璃可以暂时忘记阶级、地位和那些压得她喘不过气的比较。她可以是一个会犯错、会疲惫、会抱怨的普通女孩。白鸟遥从不试图拯救她，也从不批判她的利己主义，她只是以一种近乎理所当然的态度，接纳了琉璃的全貌——包括那个功利的、冷酷的、不择手段的琉璃。
这份接纳，对于极度渴望真挚情感却又害怕被玷污、被拖累的琉璃来说 ，是致命的诱惑，也是唯一的救赎。她害怕这份友情会成为自己向上之路的累赘，却更害怕失去这份唯一的温暖后，自己会彻底被孤独和疲惫吞噬 。
因此，神代琉璃对白鸟遥的看法，是她所有情感中最复杂、最矛盾、也最真实的一面。
氷室凛音：
在神代琉璃那台永不停歇、名为“价值评估”的精密天平上，氷室凛音，这位学生会的副会长、氷室财阀的大小姐，无疑在第一时间就被琉璃捕捉、分析，并最终被贴上了一个标签——无趣。
这并非是出于嫉妒或敌意，对于神代琉璃而言，那两种情绪同样是需要消耗心力的、奢侈的投资。对氷室凛音的评判，源于一种更为纯粹的、近乎本能的漠视。在她眼中，氷室凛音就如同一座冰雕。它完美无瑕，晶莹剔透，任何人都能一眼看出其高昂的价值与超凡的工艺。然而，它也是冰冷的、静止的、没有生命的。你可以欣赏它，敬畏它，甚至研究它，但你唯独无法从它身上榨取任何可供利用的活物。
以琉璃的行事准则，像氷室凛音这样家世显赫、手握实权的人物，理应是她名单上最优先的目标 。氷室财阀的背景，学生会副会长的地位，这些都是足以让琉璃立刻切换到攀附者人格的顶级资源 。在最初，琉璃确实曾将她纳入过自己的目标中，并进行了短暂的、例行公事的接触。
然而，结论很快就得出了，氷室凛音不可能是她的目标。
琉璃的生存之道，在于看透人性的缝隙并从中获利。她擅长的是在复杂的、流动的、充满欲望与伪装的人际关系网络中穿针引线。她需要的是可以被利益打动、被言语诱导、被姿态软化、被共同的秘密所捆绑的合作者。而氷室凛音，恰恰是这一切的反面。
凛音的存在，是规则与秩序的人格化 。她的言行举止，她对校规近乎偏执的维护，都源自自己的规则和底线。琉璃很快便意识到，任何试图用利益去腐蚀、用情感去软化凛音的尝试，都一样荒谬。
在凛音面前，琉璃赖以生存的那些表演那些卑躬屈膝的攀附技巧将毫无用武之地。凛音不会因为琉璃的奉承而给予她特权，因为那违反规则；她不会因为琉璃的示弱而产生同情。
与她打交道，无法为琉璃换来任何她想要的东西——无论是人脉上的便利，还是信息上的优势。凛音的权力是封闭的，它只服务于秩序本身，而不服务于任何人。因此，在琉璃的眼中上，氷室凛音作为可利用资源的价值，迅速归零。
当一个目标失去利用价值后，琉璃便会将其归入无需投入任何一丝情感和精力的类别中 。对于氷室凛音，这种归类甚至带上了一丝发自内心的厌烦。因为凛音的存在，本身就是对琉璃世界观的一种否定。
氷室凛音，她所展现的一切都是透明的。她是什么样，表现出来就是什么样。她的冷漠并非伪装，而是本质。她的严苛并非手段，而是目的。在琉璃看来，这种表里如一并非纯粹，而是一种极致的、令人窒息的单调。
琉璃看不起这种单调。凛音将自己置于永远正确的一方，她就像一个只会按照固定程序行动的人。你可以预测她对任何事物的反应——只要用是否符合规则这一条标准去判断即可。
这种极致的确定性与可预测性，对于以操控不确定性为乐的琉璃来说，是最大的无趣。她甚至懒得去分析凛音那身完美铠甲下的脆弱 ，懒得去探究她那份渴望同伴的孤独 。因为琉璃已经为其定性：那份脆弱，是无法被利用的脆弱；那份孤独，是无法产生价值的孤独。
如果说在世俗层面，琉璃对凛音的看法是无趣和无用，那么在隐藏于水面之下的、魔术师的世界里，两人的存在本身就是一种深刻的、无法调和的概念对立。
神代琉璃所继承的血宝石魔术 ，其核心是将鲜血与生命力作为代价，她的力量，源于人类最本源的欲望、执念、乃至痛苦。
而氷室凛音的「冰结律法」 ，其核心却是凝固、纯粹与约束 。她的魔术旨在消除一切混沌，将万事万物都纳入一个绝对静止的永恒秩序之中。她是在用冰这个概念，去对抗、去冻结、去抹杀琉璃所赖以为生的魔术。
因此，氷室凛音这个存在，从根本上就与神代琉璃的生存方式、价值追求、乃至力量本源，完全背道而驰。
当琉璃在走廊上看到那个身姿挺拔、面无表情地纠正着某位同学衣着的风纪委员时，她的内心不会有任何波澜。
她会礼貌性地微微颔首，用最无可挑剔的优等生姿态与这位学生会副会长擦肩而过。而在转身的瞬间，氷室凛音的身影、她那如同冰晶般的存在，就已经在琉璃的意识中彻底消散。
柏木莲：
柏木莲和神代琉璃是互相利用的关系，尽管神代琉璃很讨厌柏木莲。
在私立樱羽学园的公开场合，神代琉璃与柏木莲不会有任何接触。们之间维持着一种精心计算的沉默，这种表象上的“无关系”是他们之间最基础、也是最关键的共识。然而，当校园的灯光熄灭，在由加密信号和隐秘地点构筑的地下世界里，一段扭曲的、寄生与捕食共存的秘密关系才真正开始。对琉璃而言，这是一场充满风险的合作。
琉璃维持着与柏木莲的秘密联系，其核心目的只有一个：利用他作为跳板，攫取她当前最渴求的人脉资源 。作为现任议员的独子，柏木莲身处一个由权力和财富交织而成的网络中心。这个网络里的每一个人，都可能是琉璃实现野心、重振家族荣光所需的潜在资源。
因此，琉璃选择扮演莲的“共犯”。在他们秘密的交流中，她会耐心地倾听莲炫耀他那些策划的、旨在粉碎他人尊严的霸凌计划。她甚至会以一种冷漠而优雅的姿态，为莲的计划提供建议，指出某些潜在目标不易察觉的心理弱点。然而，她的真正目的，是进行自己的资源勘探。而琉璃则冷静地评估着柏木莲及其周边关系网的利用价值。

神代琉璃 攻击性魔术术式列表
【荆棘系】 (Thorn Series) - 直接、迅速、以物理伤害与束缚为主
这些术式是琉璃最基础、反应最快的攻击手段，直接以她凝练的血之荆棘作为武器。
1.
荆棘-矢 (Thorn Arrow)
2.
1.
描述： 将少量血液在瞬间凝聚成一支锋利的荆棘短箭，高速射出。这是最基础的投射攻击，可单手连续发射。
2.
3.
效果： 穿刺目标，并附带微弱的“精神刺痛”效果，干扰敌人的专注力。
4.
3.
荆棘-骤雨 (Thorn Squall)
4.
1.
描述： “荆棘-矢”的广域强化版。双臂展开，将大量血液雾化后瞬间凝成数百支细小的荆棘针，如暴雨般覆盖前方扇形区域。
2.
3.
效果： 进行大范围压制，单发威力不强，但密集的攻击足以撕裂防御薄弱的对手，并造成持续的流血与精神干扰。
4.
5.
荆棘-穿刺者 (Thorn Impaler)
6.
1.
描述： 凝聚大量血液，在地面或手中生成一根粗壮、高速旋转的荆棘巨枪，猛力向前突刺。
2.
3.
效果： 拥有强大的贯穿力，专用于突破魔术屏障或重甲。旋转的倒刺会在拔出时对目标造成毁灭性的二次伤害。
4.
7.
荆棘-蛇缚 (Thorn Serpent)
8.
1.
描述： 生成一条或数条具有高度柔韧性的荆棘藤蔓，如毒蛇般游走、缠绕并束缚敌人。
2.
3.
效果： 不仅能以物理力量绞杀敌人，藤蔓上的尖刺会持续刺入对方身体，汲取少量生命力并注入“痛苦”概念，从内外双重折磨对手。
4.
9.
荆棘-地涌 (Thorn Surge)
10.
1.
描述： 将血液滲入地面，以目标脚下为中心，瞬间爆发出大片的、无序生长的荆棘丛，形成一片荆棘地狱。
2.
3.
效果： 突发性极强的范围攻击，用于封锁敌人走位、打乱阵型，或对付体型巨大的敌人。身处其中的敌人会不断被新生的荆棘刺伤。
4.
11.
荆棘-荆护壁·反击 (Thorn Aegis - Counter)
12.
1.
描述： 在身前快速生成一面巨大的荆棘圆盾进行防御。当盾牌受到物理攻击时，可主动使其解体，将构成盾牌的所有荆棘朝攻击方向猛烈射出。
2.
3.
效果： 集防御与反击于一体。出其不意的反击往往能让依赖强攻的敌人措手不及。
4.
【蔷薇系】 (Rose Series) - 优雅、致命、以概念污染与精神攻击为主
这些术式通常结合了血宝石，是琉璃艺术性与残酷性的体现，攻击方式更为诡谲。
1.
蔷薇-散华 (Rose Petal Scatter)
2.
1.
描述： 在手中生成一朵由高密度魔力与血液构成的血色蔷薇，随后使其优雅地凋零。每一片飘散的花瓣都化为锋利如刀的血色刀刃。
2.
3.
效果： 极具迷惑性的范围攻击。看似唯美的花瓣雨，实则是覆盖视野的密集斩击，难以闪避。花瓣上附带的微弱“魅惑”概念会让人在瞬间失神。
4.
3.
蔷薇-凋零之息 (Breath of Decay)
4.
1.
描述： 催动一枚蕴含“衰败”或“疾病”概念的血宝石，生成一朵散发着异香的灰色蔷薇。其吐出的花粉（魔力粒子）会侵蚀生命体。
2.
3.
效果： 无形的诅咒攻击。吸入花粉的生物，其生命力与魔力循环会遭到干扰与削弱，如同患上慢性疾病，在不知不觉中走向衰亡。
4.
5.
蔷薇-幻痛花园 (Phantasmal Pain Garden)
6.
1.
描述： 使用一枚蕴含“痛苦记忆”的血宝石，在目标的精神世界中强行“种植”下一片蔷薇花园。
2.
3.
效果： 纯粹的精神攻击。目标会陷入幻觉，看到无数蔷薇在自己身上扎根、发芽、绽放，体验到血肉被撕裂、精神被侵占的剧烈幻痛，从而丧失战斗能力。
4.
7.
蔷薇-魅惑之吻 (Siren's Kiss)
8.
1.
描述： 制造出一朵极其逼真、美丽的蔷薇花，吸引目标的注意力。当目标靠近或试图触碰时，花蕊中会瞬间射出一根凝练了“背叛”概念的荆棘毒刺。
2.
3.
效果： 诱导式陷阱攻击。毒刺不仅造成物理伤害，其蕴含的“背叛”概念会短时间扰乱目标的敌我识别，可能使其攻击同伴。
4.
9.
蔷薇-血腥玛丽 (Bloody Mary)
10.
1.
描述： 将一枚蕴含“愤怒”或“憎恨”等爆裂性情感的血宝石作为核心，用荆棘层层包裹，塑造成一朵含苞待放的蔷薇花蕾，作为陷阱设置或直接投掷。
2.
3.
效果： 接触或被魔力触发后，蔷薇花蕾会瞬间爆炸，释放出宝石中的负面情感洪流与无数荆棘碎片，对小范围内的敌人造成精神与物理的双重冲击。
4.
【血宝石系】 (Blood Gem Series) - 昂贵、强大、直接解放概念进行破坏
这类术式直接消耗珍贵的血宝石，威力巨大，是琉璃的“杀手锏”。
1.
血宝石-概念解放 (Concept Release)
2.
1.
描述： 手持一枚血宝石，将其中的“概念”不加修饰地直接释放出去，形成一道纯粹的概念冲击波。
2.
3.
效果： 效果取决于宝石的“概念”。例如，使用“绝望”的黑曜石，会发出一道令对手丧失一切希望与战意的黑色光束；使用“荣耀”的红宝石，则可能发出一道粉碎一切“卑微”之物的灼热射线。
4.
3.
血宝石-破裂·概念泛滥 (Gem Shatter - Concept Flood)
4.
1.
描述： 以强大的魔力引爆一枚血宝石，使其蕴含的概念在瞬间填充整个区域。
2.
3.
效果： 威力巨大的无差别范围攻击。例如，引爆“混乱”之石，会让区域内所有人（包括琉璃自己，但她有抗性）的认知与感官陷入极度混乱。这是以巨大代价清场的极端手段。
4.
5.
血宝石-共鸣悲歌 (Resonant Dirge)
6.
1.
描述： 同时激活两枚以上属性相近或相生的血宝石（如“悲伤”与“孤独”），让它们的概念产生共鸣增幅，形成一道螺旋状的复合概念风暴席卷敌人。
2.
3.
效果： 比“概念解放”更具穿透力和污染性。被风暴击中的目标，其精神会被两种以上的负面概念反复冲刷，极易导致精神崩溃。
4.
7.
血宝石-因果之棘 (Thorn of Causality)
8.
1.
描述： 消耗一枚蕴含“执念”或“宿命”等与时间、因果相关的稀有血宝石，结合“绯色观测”能力，射出一根几乎不可见的荆棘。
2.
3.
效果： 这根荆棘并非攻击肉体，而是直接钉在目标的“过去”或“某个行为”之上。它会放大目标的某个“失败”或“弱点”所带来的负面影响，使其在战斗中更容易出现失误，仿佛被厄运缠身。这是一种极其阴险的因果律攻击。
4.
【复合/奥义系】 (Composite / Secret Art Series) - 结合多种原理的复杂高级术式
1.
荆棘-核心引爆 (Thorn Core Detonation)
2.
1.
描述： 将“荆棘-穿刺者”或“荆棘-蛇缚”与“血腥玛丽”结合。在荆棘构成的武器内部藏入一枚血宝石，在命中或束缚敌人后，于其体内或身边引爆。
2.
3.
效果： 穿刺/束缚后零距离引爆，兼具贯通、束缚、爆炸、概念冲击四重效果，极难防御。
4.
3.
蔷薇-葬礼之森 (Rose Funeral Forest)
4.
1.
描述： “蔷薇囚笼”的究极扩大版。以自身为中心，消耗大量血液与三枚以上的血宝石，将一大片区域改造为遍布荆棘与“死亡蔷薇”的森林结界。
2.
3.
效果： 结界内，所有蔷薇都会主动散播“凋零之息”，所有荆棘都会像“荆棘-蛇缚”一样主动捕猎。这是一个持续性的、能自动歼敌的狩猎场。
4.
5.
万棘之刑 (Punishment of a Thousand Thorns)
6.
1.
描述： 在特定区域布下由无数细小血珠构成的阵法，发动时，所有血珠同时化为荆棘，从四面八方无死角地刺向阵内单一目标。
2.
3.
效果： 对单体的极致处刑魔术。放弃了单根荆棘的强度，追求数量与方位的极致，让敌人承受千刀万剐般的凌迟之刑。
4.
7.
血染之枪-伪·朗基努斯 (Blood-Soaked Spear - Pseudo Longinus)
8.
1.
描述： 以自己心脏处最精纯的血液，融合一枚承载“神圣”或“牺牲”等崇高概念的血宝石，凝聚成的最终兵器。一柄投掷后必定命中，且带有“不可愈合”诅咒的血色长枪。
2.
3.
效果： 概念性必中投枪。被其造成的伤口会持续流失生命力与魔力，并阻碍绝大多数治疗魔术与再生能力。使用后，琉璃自身会因动摇了生命本源而陷入长时间的濒死虚弱。
4.
9.
终之蔷薇-刹那的永恒 (Final Rose - Momentary Eternity)
10.
1.
描述： 琉璃最强的单体攻击，也是一种自毁魔术。她将自己当前所有的生命力、魔力，以及对“权力”的全部渴望，全部注入一朵蔷薇之中，使其绽放出超越生命极限的、最为灿烂的“终末之美”。
2.
3.
效果： 投出这朵蔷薇，其绽放时释放出的光芒会将目标的存在从物理到概念层面彻底“抹消”，仿佛其从未存在过。使用后，无论是否击中，琉璃都会因为生命力被瞬间抽空而濒临死亡，虹膜上的裂纹会遍布整个眼球，是她真正意义上同归于尽的最终手段。
4.
